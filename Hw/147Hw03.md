## Utility class = System.Match  
เพราะ เป็นเครื่องมือทางคณิตศาสตร์แค่คำนวณ  
มีแต่ Static methods  
ไม่ต้องสร้าง object ก่อนที่จะใช้งาน
และไม่มีข้อมูลภายในที่เปลี่ยนค่าได้ เรียกใช้แล้วจบไม่ได้เก็บค่าอะไรไว้  
  
## Mutable Class = System.Collections.Generic.List<T>  
เพราะ สามารถเปลี่ยนแปลงข้อมูลภายใน objectได้ เช่น add หรือ remove และสามารถแก้ไขข้อมูลในlistเดิมได้  
ทำให้สถานะของ object เปลี่่ยนโดยไม่ต้องสร้างใหม่ เลยเป็น class ที่ state เปลี่ยนแปลงได้  
  
## Immutable Class = System.String  
เพราะ ถ้าสร้าง string แล้วค่าภายในจะไม่สามารถเปลี่ยนแปลงได้ หากมีการแก้ไขข้อความ  
อย่างเช่น ถ้าจะต่อ string ระบบจะสร้าง object ใหม่ขึ้นมาแทนของเดิม ทำให้ค่าของ object เดิมไม่เปลี่ยนแปลง  
classนี้เลยเป็นclassที่ไม่สามารถแก้ไขสถานะภายในได้
