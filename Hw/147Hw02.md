**Bhasoontara Paenphai 66102010147 ได้ Code Smell : Incomplete Library Class**
โดยความหมายของ Incomplete Library Class คือ ใช้งานจาก library class แต่คลาสนั้นยังไม่ครบถ้วนตามที่ระบบต้องการ เช่น ไม่มี method หรือ function ที่เราต้องการ ก็คือ **มันดีแต่มันยังไม่ครบสำหรับงานของเรา** ทำให้ไม่สามารถใช้งานได้ตรงตามความต้องการ ส่วนใหญ่เกิดขึ้นเมื่อระบบมีความต้องการเฉพาะทาง ทำให้ต้องเขียนโค้ดเสริมภายนอกหลายครั้ง หรือเขียนโค้ดซ้ำๆ ทำให้เกิดความซับซ้อนที่มากขึ้น และเสี่ยงต่อการเกิดbug *ยังทำให้ยากต่อการบำรุงรักษาในระยะยาว*
**ตัวอย่างโค้ดที่เกิด Code Smell**


DateTime date = DateTime.Now;

bool isWeekend =
    date.DayOfWeek == DayOfWeek.Saturday ||
    date.DayOfWeek == DayOfWeek.Sunday;

**ปัญหา (Code Smell)**
-DateTime ไม่มีเมธอด IsWeekend()

**ตัวอย่างโค้ดที่ Refactoring**


public static class DateTimeExtensions
{
    
    public static bool IsWeekend(this DateTime date) 
    {
        return date.DayOfWeek == DayOfWeek.Saturday || date.DayOfWeek == DayOfWeek.Sunday;
    }
}

DateTime today = DateTime.Now;
bool result = today.IsWeekend();

**ตัวอย่างโค้ดที่ไม่จำเป็นต้อง Refactoring**



bool isWeekend =
    date.DayOfWeek == DayOfWeek.Saturday ||
    date.DayOfWeek == DayOfWeek.Sunday;

runได้ไม่ได้ถือว่ามีปัญหาร้ายแรงเพราะใช้แค่ครั้งเดียวหรือไม่บ่อย,ไม่ก่อให้เกิดการซับซ้อน
